Situs kodegara.org

## Instalasi

```console
yarn install
```

## Pengembangan lokal

```console
yarn start
```

## Build

```console
yarn build
```

## Deploy

Menggunakan github pages:

```console
GIT_USER=<Your GitHub username> USE_SSH=true yarn deploy
```
